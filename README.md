# Traqueur de lecture

Prototype React permettant de :

- rechercher un livre par son titre ;
- ajouter ou retirer un livre des favoris ;
- marquer un livre comme "je souhaite le lire" ;
- filtrer la liste sur les favoris, les souhaits, ou l'intersection des deux.

## Lancer le projet

```bash
npm install
npm run dev
```

## Structure

- `src/data/books.json` : dataset fourni pour l'exercice
- `src/App.jsx` : logique d'etat, recherche, filtres et affichage
- `src/index.css` : styles de l'interface
